    Entity

    Entity - actor of the scene, container for behaviour components.

    Example:

    const entity = new Entity();
entity.deserialize({ name: 'hey', components: { Hello: { hello: 'world' } } });
const hello = entity.getComponent('Hello');
console.log(hello.hello);
